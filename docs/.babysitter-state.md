## Gate 1 — Story Expansion

### PRD

Sourced from `docs/chat-response-beautifier-plan.md` (pre-existing before babysitter session).

**Summary**: After every assistant stream completes, send the raw text through an isolated single-shot LLM call that cleans formatting, then commit the beautified result to the `messages` store before markdown rendering. The feature is always-on via a hardcoded default prompt. An optional Obsidian `beautifierPromptPath` setting (Sprint 2) lets power users supply their own instructions.
